# Hospital-Analytics-Dashboard
Hospital Analytics Dashboard using Power BI
Project Overview

This project presents an interactive Hospital Analytics Dashboard developed using Power BI to analyze patient demographics, treatment costs, insurance coverage, and departmental revenue. The dashboard enables healthcare administrators to monitor KPIs and derive insights for data-driven decision-making.

Business Problem

Healthcare organizations need an efficient way to monitor treatment costs, hospital revenue, insurance coverage, and departmental performance. This dashboard provides an interactive solution for analyzing these key business metrics.

Objectives


Analyze hospital revenue and treatment costs.
Compare revenue across departments.
Analyze treatment costs by age group.
Monitor insurance coverage.
Track patient demographics.
Build an interactive dashboard using Power BI.
