# 🏥 Hospital Analytics Dashboard | Power BI

## 📌 Project Overview

The **Hospital Analytics Dashboard** is an interactive Business Intelligence solution developed using **Power BI** to analyze hospital revenue, treatment costs, patient demographics, insurance coverage, and departmental performance. The project uses a **Star Schema** data model and **DAX** measures to generate meaningful business insights and support data-driven decision-making.

---

## 🎯 Business Problem

Healthcare organizations generate large volumes of patient and financial data. Without proper analysis, it becomes difficult to monitor revenue, treatment costs, insurance claims, and department performance.

This dashboard provides an interactive solution to monitor key healthcare metrics and improve operational decision-making.

---

## 🎯 Project Objectives

- Analyze Total Revenue and Treatment Cost
- Compare department-wise hospital performance
- Monitor patient demographics
- Analyze treatment cost by Age Group
- Track Insurance Coverage
- Build an interactive Power BI Dashboard
- Generate business insights using DAX

---

# 📂 Dataset

The dataset follows a **Star Schema** consisting of one Fact table and multiple Dimension tables.

### Fact Table
- Fact_Visits

### Dimension Tables
- Dimension_Patients
- Dimension_Doctors
- Dimension_Departments

# 💡 Key Business Insights

- Neurology generated the highest hospital revenue.
- Young Adults contributed the highest treatment cost.
- Most patients were covered by insurance.
- Cardiology recorded the highest average treatment cost.
- Interactive filters allow detailed analysis by department, doctor, gender, and insurance status.
